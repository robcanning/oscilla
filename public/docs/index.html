Rotula is controlled by: - SVG element IDs and attributes defining animation, paths, cues, and behaviors - Keyboard shortcuts and a minimal GUI for playback control - Cues embedded directly into the score to control timing, triggers, and logic - OSC output, synchronized to the playhead or cue events

üñºÔ∏è Design Workflow

  1. Create SVG scores in Inkscape (or similar), using long horizontal layouts
  2. Tag elements using cue namespaces (e.g., cue_pause, cue_audio_theme)
  3. Add animation using namespaces like obj2path, r_deg[...], or s[...]
  4. Load the SVG into Rotula, set duration, and go
