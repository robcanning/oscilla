Wi-Fi Router Setup for Rotula Concert Box (TP-Link Archer C6)

This guide is specifically for configuring the TP-Link Archer C6 router for use with the Rotula "Concert Box" ‚Äî a local server setup for concerts or workshops, accessed by multiple iPads or other devices without internet.

‚öôÔ∏è Configuration Checklist (Router Admin Panel)

  1. Connect a laptop via Ethernet
  2. Open a browser and go to the router admin page:
    • Usually http://192.168.0.1 or http://tplinkwifi.net
  3. Login with the default credentials (printed on the router)
  4. Follow these setup steps:

Wi-Fi Setup

  • Set SSID to something recognizable, e.g., Rotula-Stage
  • Enable both 2.4GHz and 5GHz bands
  • Set a strong Wi-Fi password
  • Disable guest network (unless intentionally used)

Internet Settings

  • You can leave WAN disconnected (offline use)
  • Do not enable captive portals or parental controls

DHCP Settings

  • Go to DHCP ‚Üí Address Reservation
  • Reserve a static IP for your Concert Box (e.g., 192.168.0.10)
  • Alternatively, manually set a static IP on the box itself

Other Recommendations

  • Set DHCP lease time to a long duration (24h or more)
  • Write down the SSID and password and tape it to the router
  • Test the IP address in a browser (e.g., http://192.168.0.10:8001)

üåê mDNS / .local Notes

  • The TP-Link Archer C6 does not support mDNS (.local) out of the box.
  • Access the Concert Box using its static IP instead of rotula.local.
  • Example: http://192.168.0.10:8001
